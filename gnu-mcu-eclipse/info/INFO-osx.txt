
This is the GNU MCU Eclipse (formerly GNU ARM Eclipse) version of RISC-V GCC for Apple OS X.

If generally follows the official releases.

The changes are minimal, and mainly consist in the packing
procedure to generate the setup/install packages (`CHANGES.txt`).

For more info and support, please read the GNU MCU Eclipe project pages from:
